# UNIT-0009 RECORDING IDENTITY FIELDS

A recording may be uniquely distinguished using:

- recording_title
- song_id
- group_id
- performer_id
- recording_date
- release_id
- label_id
- catalog_number
- matrix_number
- duration
- language
- country

Priority:

1. matrix_number
2. catalog_number
3. recording_date
4. group + song
5. performer + song

Human review remains final authority.
